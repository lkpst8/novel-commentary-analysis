# Long-Novel Commentary Framework

## Purpose

Use this reference to analyze a long novel before generating the final HTML page. The goal is to help a reader quickly understand what story the book tells, who drives it, what forces shape it, and why it leaves an impression.

## Core Analysis Targets

Always try to extract:

- `故事背景`
- `世界规则`
- `核心人物`
- `主要关系`
- `主线冲突`
- `关键支线`
- `重大转折`
- `结局落点`
- `主题表达`
- `作品气质`

These are the minimum ingredients for a useful long-novel commentary page.

## Reading Passes

### Pass 1: Establish the story frame

Answer:

- 故事发生在什么时代、地点、制度或社会环境里
- 主人公是谁
- 最核心的矛盾是什么
- 小说最强烈的情绪底色是什么

Output:

- One-sentence premise
- Story frame paragraph
- Early major-character list

### Pass 2: Build the main line

Split the central story into 5 to 10 stages.

For each stage, record:

- 起点状态
- 冲突目标
- 阻碍力量
- 转折事件
- 对后续剧情的推动

This is the backbone of the `主线剧情` section.

### Pass 3: Separate the subplots

For every important subplot, ask:

- 它围绕谁展开
- 它补充了什么人物、主题或背景
- 它是否改变了主线走向
- 它是独立支线，还是主线的侧面推进

Only keep subplots that truly affect story understanding. Do not turn minor incidents into fake “major lines”.

### Pass 4: Map character functions

For each major character, record:

- 身份/定位
- 与主角的关系
- 核心诉求
- 主要矛盾
- 关键转折
- 最终落点

Useful dramatic functions:

- `主角`
- `关键对手`
- `同盟/伙伴`
- `情感支点`
- `制度压力来源`
- `镜像人物`
- `支线推动者`

### Pass 5: Extract theme from repetition

Theme should come from repeated patterns such as:

- repeated institutions
- repeated emotional wounds
- repeated class or power asymmetry
- repeated impossible choices
- repeated sacrifice, betrayal, concealment, duty, desire, or longing

Good theme prompts:

- 小说最反复地让人物付出代价的东西是什么
- 什么样的欲望在这个故事里最危险
- 哪种制度、家庭结构或关系秩序塑造了命运
- 故事最终保留了什么，又击碎了什么

## Anti-Hallucination Discipline

Use these rules throughout:

- If the text does not state it, do not state it as fact.
- If the text only implies it, label it as `分析判断`.
- If the source is incomplete, say `材料不足`.
- If a relation or motive is unclear, keep it open rather than forcing certainty.
- If chronology is broken in the source, describe the break instead of silently fixing it.
- If local source text exists, do not replace source reading with web summaries or plot recaps from outside sites.
- If you lack enough local material, say `材料不足` instead of importing missing plot from external summaries.

## Anti-Omission Discipline For Huge Novels

When the novel is very long, missing important material becomes as dangerous as hallucinating it.

Before writing the final HTML, check:

- which packets contribute to the main line
- which packets mainly update subplots
- where each major character first becomes important
- where each major character changes direction
- where the background or world rules materially shift
- whether the ending reflects the late phases, not just the opening premise

If you cannot answer those questions, you are not ready to finalize.

## What Makes the Final Page Useful

A good final page lets a reader:

- grasp the whole story fast
- identify the major people and pressures
- understand the main line without getting lost in detail
- see how subplots connect back to the whole
- understand the ending, theme, and emotional residue

If a section does not improve one of those goals, simplify it.
